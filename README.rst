Bayesian Filtering and Calibration Toolkit (BayesFilt)
=======================================================


This toolkit implements various Kalman Filtering algorithms and
Bayesian statistical tools.


Installation
--------------

Clone the GitHub repository on local machine,
cd into the bayesfilt directory and do:

.. code-block:: bash

    pip install .

For editable install (for development purpose), do

.. code-block:: bash

    pip install -e .
